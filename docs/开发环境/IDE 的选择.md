# IDE 的选择

- Arduino IDE

    - Arduino、ESP32、STM32 等

- Thonny

    - Raspberry Pi 系统里编写，VNC 连接编写

    - ESP32

- VS code：中大型项目，免费，已经足够智能强大

    - PlatformIO：几乎所有框架、芯片平台

    - Remote - SSH：远程开发 Raspberry Pi

    - 其它芯片家插件

- Clion：中大型项目，更加智能的代码提示，体验感好

    - PlatformIO：几乎所有框架、芯片平台

    - STM32Cube 创建框架，Clion 编写代码
